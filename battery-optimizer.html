🧠 脳にいいアプリ

バッテリー最適化無効化ガイド
⚙️ バッテリー最適化設定を開く

Samsung 設定手順

1. 設定 アプリを開く
2. バッテリーとデバイスケア をタップ
3. バッテリー をタップ
4. バックグラウンドアプリの使用量制限 をタップ
5. 脳にいいアプリ を探して 制限しない を選択

Sony 設定手順

1. 設定 アプリを開く
2. バッテリー をタップ
3. アダプティブバッテリー をタップ
4. 制限されたアプリ を確認し、脳にいいアプリ が含まれていないことを確認

Google Pixel 設定手順

1. 設定 アプリを開く
2. バッテリー をタップ
3. バッテリー最適化 をタップ
4. 「すべてのアプリ」に切り替え
5. 脳にいいアプリ をタップして 最適化しない を選択

Xiaomi 設定手順

1. 設定 アプリを開く
2. アプリ をタップ
3. アプリを管理 をタップ
4. 脳にいいアプリ を検索して選択
5. バッテリーセーバー をタップして 制限なし を選択

Huawei 設定手順

1. 設定 アプリを開く
2. バッテリー をタップ
3. アプリ起動管理 をタップ
4. 脳にいいアプリ を探して手動管理をオンにする
5. 「自動起動」「関連起動」「バックグラウンド動作」を全てオンにする

OPPO 設定手順

1. 設定 アプリを開く
2. バッテリー をタップ
3. アプリクイック凍結 または スリープスタンバイ最適化 をタップ
4. 脳にいいアプリ を探して除外リストに追加

4. 「すべてのアプリ」を選択
5. 脳にいいアプリ をタップして 最適化しない を選択

その他のメーカー 設定手順

1. 設定 アプリを開く
2. バッテリー または 電源管理 を探す
3. バッテリー最適化 または アプリの電源管理 を開く
4. 脳にいいアプリ を探してタップ
5. 最適化しない または 制限なし を選択
📝 注意: メーカーやAndroidバージョンによって設定項目名が異なる場合があります。類似の項目名を探してください。設定後は、脳にいいアプリが正常に動作することをご確認ください。
